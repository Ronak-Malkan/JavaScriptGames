Type It To Lose It

It's simple really. Words will start dropping down from the top. You have to simply type them and they will disappear. Once you start typing a word, you have to complete it before you can start typing another word. If any word reaches the bottom, game is over. For every word, you will get points equivalent to the length of the word.

You can change word fall speed, and word spawn speed below.
To change the color scheme, i.e. light/dark, click anywhere on the screen.
